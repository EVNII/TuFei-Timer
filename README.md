# 土肥杯计时器

基于SSIM对比相似度删除传送时间导致的设备差异。

</div>


## 简介

> [!IMPORTANT]
> 希望不再有设备导致的差异。

**土肥杯计时器** 是基于图像识别技术、在线捕获原神窗口通过SSIM对比传送画面相似度来辅助去掉传送时间的计时。目的是为了减轻不同设备的传送时间差异。
